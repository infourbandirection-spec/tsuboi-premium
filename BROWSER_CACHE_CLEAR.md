# ブラウザキャッシュクリア手順

## 問題の症状
- 管理画面で変更が反映されない
- 「予約ID」が「応募ID」に変わっていない
- 検索結果で氏名・電話番号が表示されている（削除されたはず）
- 新機能が表示されない

## 原因
ブラウザが古いバージョンの JavaScript ファイルをキャッシュしているため

## 解決方法

### 方法1: ハードリフレッシュ（推奨）

#### Windows/Linux
```
Ctrl + Shift + R
または
Ctrl + F5
```

#### Mac
```
Command + Shift + R
または
Command + Option + R
```

#### スマートフォン
- **iPhone (Safari)**: 設定 → Safari → 詳細 → Webサイトデータ → すべてを削除
- **Android (Chrome)**: 設定 → プライバシーとセキュリティ → 閲覧履歴データの削除 → キャッシュされた画像とファイル

### 方法2: ブラウザ開発者ツールを使用

#### Chrome/Edge
1. F12キーを押して開発者ツールを開く
2. ネットワークタブを選択
3. **「Disable cache」にチェック**を入れる
4. 開発者ツールを開いたままページをリロード

#### Firefox
1. F12キーを押して開発者ツールを開く
2. ネットワークタブを選択
3. 歯車アイコン → **「キャッシュを無効化」にチェック**
4. 開発者ツールを開いたままページをリロード

### 方法3: キャッシュとCookieを削除

#### Chrome/Edge
1. `Ctrl + Shift + Delete` (Windows) または `Command + Shift + Delete` (Mac)
2. **「期間」を「全期間」**に設定
3. **「キャッシュされた画像とファイル」にチェック**
4. 「データを削除」をクリック
5. ブラウザを再起動
6. https://passurt24.pages.dev/admin に再度アクセス

#### Firefox
1. `Ctrl + Shift + Delete` (Windows) または `Command + Shift + Delete` (Mac)
2. **「期間」を「すべての履歴」**に設定
3. **「キャッシュ」にチェック**
4. 「今すぐ削除」をクリック
5. ブラウザを再起動
6. https://passurt24.pages.dev/admin に再度アクセス

#### Safari (Mac)
1. Safari → 環境設定 → 詳細
2. **「メニューバーに"開発"メニューを表示」にチェック**
3. 開発 → **キャッシュを空にする**
4. ブラウザを再起動
5. https://passurt24.pages.dev/admin に再度アクセス

### 方法4: シークレット/プライベートモードで確認

#### Chrome/Edge
```
Ctrl + Shift + N (Windows)
Command + Shift + N (Mac)
```

#### Firefox
```
Ctrl + Shift + P (Windows)
Command + Shift + P (Mac)
```

#### Safari
```
Command + Shift + N
```

シークレットモードでは常に新しいファイルを読み込みます。

## 確認方法

### 更新が正しく反映されているか確認
1. https://passurt24.pages.dev/admin にアクセス
2. 管理者ログイン
3. 以下を確認：
   - ✅ タブ名が「応募一覧」になっている（「予約一覧」ではない）
   - ✅ テーブルヘッダーが「応募ID」になっている（「予約ID」ではない）
   - ✅ 「予約検索」タブで検索すると、氏名・電話番号が表示されない
   - ✅ 検索結果に生年月日、ふりがな、メールアドレスが表示される

### ファイルのバージョン確認
開発者ツール（F12）→ Consoleタブで以下を実行：
```javascript
// admin.jsが最新か確認
fetch('/static/admin.js').then(r => r.text()).then(t => console.log(t.includes('応募ID') ? '最新版' : '古いバージョン'))
```

**結果**:
- 「最新版」と表示 → 正常
- 「古いバージョン」と表示 → ハードリフレッシュが必要

## トラブルシューティング

### Q: ハードリフレッシュしても変わらない
**A**: 以下を試してください：
1. ブラウザのすべてのタブを閉じる
2. ブラウザを完全に終了
3. ブラウザを再起動
4. https://passurt24.pages.dev/admin に再度アクセス

### Q: 別のPCでは正しく表示される
**A**: 特定のPCのブラウザキャッシュ問題です。そのPCで方法3（キャッシュとCookieを削除）を実行してください。

### Q: スマートフォンで反映されない
**A**: スマートフォンは特にキャッシュが強力です：
1. ブラウザを完全に終了（マルチタスクから削除）
2. 設定からキャッシュを削除
3. スマートフォンを再起動
4. ブラウザを起動して再度アクセス

### Q: それでも反映されない場合
**A**: 以下の情報を提供してください：
- ブラウザ名とバージョン（Chrome 120、Safari 17など）
- OS（Windows 11、macOS Ventura、iOS 17など）
- エラーメッセージのスクリーンショット
- 開発者ツールのConsoleタブのエラーログ

### キャッシュ有効期限
Cloudflare Pagesはデフォルトで以下のキャッシュ設定：
- **HTML**: キャッシュなし（常に最新）
- **JavaScript/CSS**: 最大1時間キャッシュ
- ブラウザキャッシュ: ブラウザ設定による

**注意**: ハードリフレッシュを行わないと、最大1時間古いファイルが表示される可能性があります。
